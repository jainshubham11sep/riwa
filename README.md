# Riwa India — Next.js Website

A minimalist, responsive website for **Riwa India**, a Jaipur-based textile manufacturing and export house.

## Tech

- Next.js 14 (App Router)
- Tailwind CSS
- Pure CSS animations (no heavy client libs)
- Images sourced from Unsplash (royalty-free) depicting garment manufacturing, block printing, and textile factories

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Pages

- `/` — Home
- `/about` — About Riwa India
- `/products` — Product categories overview
- `/products/[category]` — Womenswear, Menswear, Kidswear, Accessories, Home Furnishings, Printed Fabrics
- `/manufacturing` — Factory setup & manufacturing capabilities
- `/contact` — Contact / inquiry form
- `/terms`, `/privacy`, `/refund`, `/shipping` — Policy pages

## Build

```bash
npm run build
npm start
```
